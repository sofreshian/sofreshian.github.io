Array HOF

  1. Html 생성

아래 배열을 사용하여 html을 생성하는 함수를 작성시오

  1. forEach를 이용해서 풀기

​ ✓ 템플릿 리터럴 내에서 표현식 삽입을 통해 간단히 문자열 삽입이 가능하다. 이는 문자열 연결 연산자보다 가독성이 좋고, 간편하게 문자열 조합도 가능함

​ ✓ 표현식의 평가 결과가 문자열이 아니더라도 문자열로 타입이 강제 변환됨

  1. map을 이용해서 풀기

​ ✓ map은 forEach와 다르게 새로운 배열을 만들어서 반환해주기 때문에 빈 문자열을 만들어서 넣어줄 필요는 없다.

​ ✓ 객체 디스트럭처링 할당은 객체에서 프로퍼티 키로 필요한 프로퍼티 값만 추출하여 변수에 할당하고 싶을 때 유용하다


  1. 특정 프로퍼티 값 추출

요소의 프로퍼티(id, content, completed)를 문자열 인수로 전달하면 todos의 각 요소 중, 해당 프로퍼티 값만 추출한 배열을 반환하는 함수를 작성

(단, for문이나 Array#forEach는 사용하지 않도록 하자)

✓ 대괄호 연산자내 따옴표가 없는 이유는 입력되는 값이 문자열이기 때문이다.


  1. 프로퍼티 정렬

✓ sort 메서드는 기본적으로 오름차순으로 요소를 정렬한다.

✓ 그러나, sort 메서드의 기본 정렬은 유니코드 포인트의 순서를 따른다. 숫자의 경우도 문자열로 변환하여 유니코드 포인트의 순서를 비교하기 때문에 의도와 다르게 결과가 나올 수 있다.

✓ 숫자의 경우 => (-) 산술연산으로 비교함수를 만들어 를 인수로 전달 / 문자의 경우 => 비교 연산을 사용


  1. 새로운 요소 추가

새로운 요소(예를 들어 {id: 4, content: ‘Test’, completed: false })를 인수로 전달하면 todos의 선두에 새로운 요소를 추가하는 함수를 작성하라.

(단, Array#unshift 나 Array#push는 사용하지 않도록 하자)


  1. 특정 요소 삭제

todos에서 삭제할 요소의 id를 인수로 전달하면 해당 요소를 삭제하는 함수를 작성하라


  1. 특정 요소의 프로퍼티 값 반전

todos에서 대상 요소의 id를 인수로 전달하면 해당 요소의 completed 프로퍼티 값을 반전하는 함수를 작성하라

( 기존 객체의 특정 프로퍼티를 변경 / 추가하여 새로운 객체를 생성하려면 Object.assign 또는 스프레드 문법을 사용한다. )


  1. 모든 요소의 completed 프로퍼티 값을 true로 설정

todos의 모든 요소의 completed 프로퍼티 값을 true로 설정하는 함수를 작성하라

( 기존 객체의 특정 프로퍼티를 변경 / 추가하여 새로운 객체를 생성하려면 Object.assign 또는 스프레드 문법을 사용한다. )


  1. Completed 프로퍼티 값이 true인 요소의 갯수 구하기

todos에서 완료(completed: true)한 할일의 갯수를 구하는 함수를 작성하라

단, for문, Array#forEach는 사용하지 않도록하자


  1. Id 프로퍼티의 값 중에서 최대값 구하기

todos의 id 프로퍼티 값 중에서 최대값을 구하는 함수를 작성하라
