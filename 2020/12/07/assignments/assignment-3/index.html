  1. 정렬(sorting) 이란?

    • 정렬(sorting): 어떤 데이터들이 주어졌을 때 이를 정해진 순서대로 나열하는 것
    • 정렬은 프로그램 작성시 빈번하게 필요로 함
    • 다양한 알고리즘이 고안되었으며, 알고리즘 학습의 필수

  1. 버블 정렬(bubble sort) 란?

    • 두 인접한 데이터를 비교해서, 앞에 있는 데이터가 뒤에 있는 데이터보다 크면, 자리를 바꾸는 정렬 알고리즘
    • 데이터가 4개일 때, 로직을 이해해보자

    Ex) data-list = [1, 9, 3, 2]

    첫번째 로직

    ​ 1과 9 비교, 자리바꿈 없음 [1, 9, 3, 2]

    ​ 9와 3비교, 자리바꿈 [1, 3, 9, 2]

    ​ 9와 2비교, 자리바꿈 [1, 3, 2, 9]

    두번째 로직

    ​ 1과 3비교, 자리바꿈 없음 [1, 3, 2, 9]

    ​ 3과 2비교, 자리바꿈 [1, 2, 3, 9]

    ​ 3과 9비교, 자리바꿈 없음 [1, 2, 3, 9]

    세번째 로직

    ​ 1과 2비교, 자리바꿈 없음[1, 2, 3, 9]

    ​ 2와 3비교, 자리바꿈 없음[1, 2, 3, 9]

    ​ 3과 9 비교, 자리바꿈 없음[1, 2, 3, 9]

  2. 알고리즘 구현

    특이점 확인!!

    • n개의 리스트가 있는 경우 최대 n-1 번 로직 적용
    • 로직을 1번 적용할 때마다 가장 큰 숫자가 뒤에서부터 1개씩 결정
    • 로직이 경우에 따라 일찍 끝날 수도 있다. 따라서 로직을 적용할 때 한 번도 데이터가 교환된 적이 없다면 이미 정렬된 상태이므로 더 이상 로직을 반복 적용할 필요가 없다.

  1. 알고리즘 분석

    반복문이 두개 O(n2)

    최악의 경우, n*(n-1)/ 2

    완전 정렬이 되어 있는 상태라면 최선은 O(n)
